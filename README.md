# YAO-MCP

[![Rust 1.90+](https://img.shields.io/badge/rust-1.90%2B-orange.svg)](https://www.rust-lang.org/)

**YAO-MCP**, _Yet, another Obsidian MCP_, provides AI agents with **specialized tools** for working with your Obsidian vault.

## Tools

### File Operations
- `read_note` — Get note content
- `write_note` — Write a note
- `modify_note` — Update a note
